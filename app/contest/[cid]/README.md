# Contest Detail Page

## Description
Contest page showing information, progress tracking (how good the problems are solved), and problem listings with submission history.

## Components
- **ContestPage**: Main container managing contest data and progress
- **ProblemItem**: Problem display with status indicators and submission history
- **Progress Bar**: Visual representation of contest completion

## User Interactions
- Click problems to start solving
- View submission details (status, score)
- Track overall contest progress and scoring of other people
