# User Profile Page
Keep in mind that this page is not connected to the backend, which needs to be developed.

## Description
User dashboard displaying personal statistics, achievements, and recent activity in the coding platform.

## Components
- **UserAccount**: Main profile component managing user data display
- **StatCard**: Reusable component for displaying statistics with icons
- **Activity List**: Recent user activities with dynamic icons

## Key Features
- **User Statistics**: Competitions participated, problems solved, and points acquired
- **Rank Display**: User's current ranking with badge styling
- **Recent Activity Tracking**: Timeline of user actions (participated, solved, created)
- **Navigation Actions**: Quick access to create contests/problems and view competitions *(only staff can create)*
- **Dynamic Icon Mapping**: Activity-specific icons (trophy, checkmark, folder, etc.)

## User Actions
- Create new contests or problems *(only available for staff)*
- View personal competition history
- Track progress and achievements
