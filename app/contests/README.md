# Contests Page

## Description
A comprehensive contest browsing and filtering page that displays all available coding contests.

## Components
- **ContestsPage**: Main page component that manages contest data and filtering
- **FilterSection**: Sidebar component for contest filtering options
- **ContestCard**: Individual contest display component with hover effects

## Key Features
- **Dynamic Contest Loading**: Fetches contests from API with error handling
- **Advanced Filtering And Searching**: Filter by category, difficulty, and location ON DEVICE!!!

## Filtering Options
- **Category**: Algorithms, Dynamic Programming, Data Structures
- **Difficulty**: Easy, Normal, Hard
- **Location**: Remote, Burnaby, Surrey, Vancouver
